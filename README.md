# Excel Data Cleaning Guide

An interactive web-based guide designed to help data analysts learn essential data cleaning techniques in Microsoft Excel.

This project provides step-by-step explanations, practical examples, and interactive demos to simulate real-world data cleaning workflows.

## 🌐 Live Demo

https://hanafidias.github.io/excel-data-cleaning-guide

---

## 🧹 Techniques Covered

Explore **15 essential data cleaning techniques** organized into 6 categories:

### 🔁 Duplicate Handling
- Remove Duplicates
- Highlight Duplicates using Conditional Formatting
- Detect duplicates using COUNTIF

### ✍️ Text Cleaning
- TRIM
- PROPER / UPPER / LOWER
- SUBSTITUTE & Find and Replace

### 📐 Data Structuring
- Text to Columns
- Flash Fill
- Merge & Split Columns

### ❓ Missing Data Handling
- Find blank cells (Go To Special)
- Fill missing values
- Audit using ISBLANK & COUNTA

### ✅ Data Validation
- Data Validation (dropdown lists)
- Consistency checks
- Unique value validation

### 🎨 Formatting & Data Types
- Convert text to numbers
- Standardize date formats
